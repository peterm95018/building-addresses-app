Developers

The data in this site includes all campus building addresses as of Winter 2016.

These addresses are for wayfinding and emergency response purposes only. Please continue to use your 1156 High St address for deliveries.

Developers may query and receive results in JSON format, address and locative data for campus buildings and mail stops. This is a work in progress and will continue to evolve. Suggestions are welcome.

REST API
URL HTTP Verb Result
http://building-addresses-api.ucsc.edu/api/locations GET Returns all entries
http://building-addresses-api.ucsc.edu/api/locations POST New entry created. (requires authentication)
http://building-addresses-api.ucsc.edu/api/locations/:id GET Returns single location
http://building-addresses-api.ucsc.edu/api/locations/:id PUT Updates an existing entry. (requires authentication)
http://building-addresses-api.ucsc.edu/api/locations/:id DELETE Deletes an existing entry. (requires authentication)
Tech Notes

This version is running as a NodeJS app on the backend via ExpressJS. The backend includes a MongoDB collections for locations, building aliases, mail stops and non-building areas (parking lots, meadows, etc.). The desktop and mobile app is written in AngularJS.

Future Development

We'll add campus parking lots and addresses as time allows. Since that isn't used besides safety services, we'll try to hide them from the average consumer.
